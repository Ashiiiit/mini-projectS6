Expense Tracker

A full-stack, AI-powered financial management system that transforms passive expense logging into "actionable financial intelligence".
 🚀 Key Features:
 * Smart Debt Settlement**: Uses a **Graph Greedy Algorithm** to minimize group transactions.
* Financial Health Advisor**: An **Expert System** using weighted heuristics to provide actionable spending grades (A-F).
* High-Performance UI**: Built with **React, Tailwind, and Recharts** for dynamic, real-time data visualization.
* Robust Backend**: Asynchronous **FastAPI** engine with ACID-compliant **SQLite** storage.

## 🛠 Tech Stack

* **Frontend**: React.js, Vite, Tailwind CSS, Recharts
* **Backend**: Python, FastAPI
* **Database**: SQLite

## 🧠 How it Works

1. **Input**: User logs transaction via the React frontend.
2. **Process**: Backend runs the **Heuristic Engine** (scoring) or **Graph Greedy Logic** (settlement).
3. **Storage**: Data is committed to the **SQLite** database.
4. **Update**: Frontend instantly redraws **SVG charts** via Recharts.
